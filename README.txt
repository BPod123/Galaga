This is Galaga. The game has 6 Levels that increase in difficulty as times goes on.
The increased difficulty results from the game running faster at each level forcing the player to make decisions quicker.
Completing a level earns the player an extra life.

GBA Controls:
Start - Begin game / Reset game
Up Arrow - Shoot
Left Arrow - Move Left
Right Arrow - Move Right

Keyboard Controls:
Enter - Begin game / Reset game
Up Arrow - Shoot
Left Arrow - Move Left
Right Arrow - Move Right
